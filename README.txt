CÀI ĐẶT MÔI TRƯỜNG
Sử dụng Pyton 3.6.8
Cài đặt các thư viện: dùng lệnh pip install -r requirements.txt qua Command Prompt để tiến hành cài đặt( trong quá trình chạy chương trình nếu có xảy ra lỗi thiếu thư viện thì chạy lệnh pip install <tên thư viện bị thiếu> để cài đặt)
Chạy file main.py: dùng lệnh python main.py để chạy chương trình

HƯỚNG DẪN SỬ DỤNG TOOL:

- Sau khi chạy chương trình, chương trình sẽ yêu cầu chọn hình ảnh có trong thư mục Imgs.
- Chương trình sẽ xuất ra hình ảnh, click vào ảnh đó sẽ xuất ra đa giác nhận diện đối tượng.
- Khi lia chuột vào một đỉnh của đa giác, đỉnh đó sẽ đổi màu nếu như lia trúng vào tọa độ của đỉnh đó.
- Có thể sửa tọa độ của đỉnh bằng cách click vào 1 đỉnh và di chuyển đến tọa độ mong muốn.
